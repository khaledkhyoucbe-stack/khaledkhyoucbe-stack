# ✨ منصة مدرستي — نظام إدارة متكامل ✨

نظام **متكامل واحترافي** لإدارة المدارس والمؤسسات التعليمية.

يوفر التطبيق إدارة شاملة للطلاب، الدرجات، الحضور، الرسوم والمصروفات، ويدعم تعدد اللغات (العربية 🇸🇦 | الإنجليزية 🇬🇧 | الفرنسية 🇫🇷 | التركية 🇹🇷).

## 🎯 المميزات الرئيسية:
- 📚 إدارة السجلات الأكاديمية والمالية المتقدمة
- 🌐 دعم تعدد اللغات الكامل (4 لغات)
- ☁️ نشر سحابي مباشر عبر Railway
- 🔐 نظام تحقق من البريد الإلكتروني المحسّن
- 💾 قاعدة بيانات قوية (SQLite / PostgreSQL)
